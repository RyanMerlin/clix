# Release v0.5.2

This release updates the public-facing docs and badges, and makes the Linux jail escape tests skip gracefully on GitHub runners where AppArmor blocks unprivileged user namespaces.

Highlights:

- README now surfaces the CI badge and latest-release badge
- Public release docs point at `v0.5.2`
- Linux jail escape tests skip instead of aborting when the host kernel blocks user namespaces

Build artifacts for this tag are published by GitHub Actions as platform-specific release tarballs with checksum, SBOM, and provenance sidecars.
