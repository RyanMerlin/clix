# clix v0.5.0

This release is the Rust 2024 migration and cleanup pass for the workspace.
It is the first public-facing release of the repository after the edition bump.

## Highlights

- Bumped the workspace to `edition = "2024"` and `resolver = "3"`.
- Raised the published crate versions to `0.5.0`.
- Cleaned up the remaining `clippy` warnings across `clix-cli`, `clix-core`, `clix-serve`, and `clix-broker`.
- Updated the Linux test harness to the new `ctor` 1.x syntax.
- Removed an unused direct `rand` dependency and refreshed the lockfile.

## Verification

The release tree was validated with:

- `cargo fmt --all`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo test --workspace --all-features`

## Notes

Rust 2024 is the latest stable edition used by this repository. If you are updating
local tooling or examples, prefer the current docs under `docs/` and the top-level
`README.md` over older roadmap milestones.
